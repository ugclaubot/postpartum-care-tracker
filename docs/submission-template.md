# Result Submission Template

Paste results in this style so they can be added quickly:

```text
Date: 2026-06-12
Days postpartum: 42
Beta-hCG: 98 IU/L
BP: 122/78
Temperature: 36.8 C
Hb: 11.2 g/dL
Ferritin: 22 ng/mL
TSAT: 18%
Fasting glucose: 92 mg/dL
OGTT 2h: 128 mg/dL
TSH: 2.8 mIU/L
EPDS: 7
Symptoms: tiredness, low milk supply
Notes: breastfeeding, C-section healing okay
```

Urgent symptoms should be written plainly, for example:

```text
Severe headache, vision change, BP 162/112
```
